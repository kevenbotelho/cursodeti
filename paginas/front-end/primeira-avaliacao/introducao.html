Introdução ao Desenvolvimento Web

Entenda Melhor

HTML é uma linguagem de marcação, onde seu objetivo é dar significado à textos, como dizer que determinada coisa é um título, um parágrafo etc.

O que é HTML

HTML, da sigla HiperText Markup Language organiza todo o conteúdo da página, como títulos, parágrafos, imagens etc. Pense no HTML como caixas que colocam os conteúdos em seus lugares certos para que apareçam organizados na tela. 

<body>

    <h1>Bem-vindo ao Meu Site!</h1>
    <h2>Sobre Este Site</h2>
    <p>Este site é um exemplo simples para demonstrar como usar HTML para estruturar uma página.A formatação básica é aplicada com CSS para melhorar a aparência.
