DESENVOLVIMENTO MOBILE

O desenvolvimento mobile refere-se ao processo de criação de aplicativos e softwares que são especificamente projetados para rodar em dispositivos móveis, como smartphones, tablets e wearables (dispositivos vestíveis). Esse tipo de desenvolvimento envolve diversas etapas, desde o planejamento e design até a programação, testes e lançamento do aplicativo.

Entenda Melhor

Desenvolvimento mobile é o processo de criar aplicativos para celulares e tablets. Esses aplicativos podem ser jogos, redes sociais, apps de banco, entre outros.

Dispositivos Móveis

Definição e origem

A telefonia móvel foi criada em 1973 por Martin Cooper, revolucionando a maneira como as pessoas se comunicam, permitindo chamadas de qualquer lugar e a qualquer momento.

Evolução dos celulares

Os celulares passaram por grandes evoluções: miniaturização, melhor qualidade de sinal (2G, 3G, 4G, 5G), novas funcionalidades (câmera, GPS, NFC), e redução de preço, tornando-os acessíveis a mais pessoas.

Sistemas Operacionais para Dispositivos Móveis

Fabricantes de celulares e SOs

Principais fabricantes: Motorola, Nokia, LG, Samsung, Apple, BlackBerry.

Cada fabricante adota ou desenvolve seu próprio Sistema Operacional (SO), que diferencia seus aparelhos.

Plataformas mais usadas

Android, iOS, Windows Phone, BlackBerry OS, Symbian. Cada uma com características específicas.

Diferenças entre as plataformas

Ferramentas de desenvolvimento e linguagens de programação são os principais fatores que diferem cada uma.

Conceitos de Sistemas Computacionais Móveis

O que são sistemas móveis?

São sistemas que podem ser movidos fisicamente ou utilizados durante o movimento.

  • Monitoramento de energia para evitar perda de dados.
  • Armazenamento local e remoto.
  • Sincronização de dados com outros sistemas.

Exemplos: sistemas para palmtops, celulares, tablets.

Breve Histórico da Computação Móvel

Primeiros dispositivos móveis

Em 1992, a Apple lançou o Newton Message PAD. Em 1996, a U.S. Robotics lançou os Palm Pilots, que dominaram o mercado de PDAs, atingindo 80% do mercado mundial.

Windows CE e Pocket PC

Surgiram em 1996 e ganharam relevância com o lançamento do Windows CE 3.0 e a plataforma Pocket PC em 2000.

Symbian OS

Desenvolvido pela empresa Symbian (fundada pela Nokia), foi um marco no desenvolvimento de sistemas operacionais móveis para smartphones.

Tablets

O conceito de tablet foi criado nos anos 60 por Alan Kay, com o nome de DynaBook. Surgiram oficialmente no final dos anos 90 e início dos anos 2000, como o Fujitsu Stylistic 2300.

Evolução das Redes Móveis 🔗

1ª Geração (1G)

Introduzida com o Motorola DynaTAC 8000X em 1973, era baseada em redes analógicas.

2ª Geração (2G)

Em 1990, as redes digitais surgiram, com o avanço da rede GSM, que melhorou a transmissão de voz e reduziu custos.

3ª Geração (3G)

Em 2001, a rede 3G permitiu maior velocidade de dados e introduziu a videoconferência.

4ª Geração (4G)

Introduzida em 2009, oferecia uma velocidade 10 vezes superior ao 3G.

Desafios da Mobilidade

Custo de hardware

Certos dispositivos móveis exigem hardware mais caro, especialmente para aplicações específicas.

Baixo poder de processamento

Para economizar energia e aumentar a autonomia, os dispositivos móveis sacrificam parte do poder de processamento.

Infraestrutura de comunicação limitada

As redes (Wi-Fi, GSM, GPRS, EDGE) ainda não são suficientemente confiáveis para todas as aplicações.

Interoperabilidade

A integração de sistemas móveis com sistemas já existentes nas empresas é um grande desafio, pois os sistemas legados não foram projetados para se comunicar com dispositivos móveis.

Linguagens de Programação para Desenvolvimento Mobile

Java (JavaME)

Líder entre os programadores, orientada a objetos e compatível com diversas plataformas, especialmente Android.

C

Comparável ao latim das linguagens de programação, é a base de muitas outras linguagens, usada para sistemas que exigem cálculos pesados ou gráficos complexos.

C++

Derivada do C, usada para uma infinidade de funções, inclusive em softwares como Photoshop e Microsoft Office.

Python

Uma linguagem simples, com sintaxe clara, usada para desenvolvimento rápido e em aplicações populares como Instagram.

Objective-C

Usada pela Apple antes da introdução da Swift, sendo a base dos aplicativos iOS.

Swift

A nova linguagem da Apple, focada em otimizar o desenvolvimento para iOS.

C#

A principal linguagem para desenvolvimento no ecossistema Microsoft, essencial para Windows Mobile.

Desenvolvimento Cross-Plataforma

MoSync

Suporta várias linguagens como C/C++ e plataformas como Symbian e Android, permitindo o desenvolvimento para múltiplos sistemas operacionais.

Whoop

Uma interface drag-and-drop que facilita o desenvolvimento para sistemas como Windows Mobile, Symbian, Android e iOS.

Appcelerator (Titanium)

Usa tecnologias web como HTML5, CSS3 e JavaScript para criar aplicativos compatíveis com várias plataformas.

Rhomobile

Uma plataforma integrada baseada em Ruby que permite desenvolver para múltiplos sistemas.

PhoneGap

Usa HTML, JavaScript e CSS3, permitindo o acesso a componentes de hardware como GPS e acelerômetro.

Adobe Air

Utiliza as extensões AIR da Adobe para acessar recursos nativos dos sistemas operacionais.

Ionic

Focado em performance e design, utiliza AngularJS e TypeScript para criar aplicativos móveis que se aproximam das aplicações nativas.

Xamarin

Baseada em C#, permite desenvolver para iOS e Android com uma única base de código.

React Native

Utilizado por grandes empresas como Facebook e Instagram, o React Native permite o desenvolvimento ágil de aplicativos multiplataforma.
