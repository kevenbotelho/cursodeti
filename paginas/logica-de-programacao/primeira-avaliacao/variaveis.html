Lógica de Programação - 02

TIPOS DE VARIÁVEIS

Antes de começarmos a falar sobre programação, precisamos entender duas coisas importantes: constantes e variáveis. Uma constante é um valor que nunca muda, como o número PI, que sempre é o mesmo. Já uma variável pode mudar, como quando guardamos o resultado de uma conta. Vamos explorar esses conceitos básicos antes de irmos mais fundo na programação.

Entenda Melhor

Elas permitem que os programas armazenem números, textos ou outras informações importantes para serem usadas em cálculos ou decisões dentro do software.

Variável

Elas são como caixas de armazenamento na memória do computador, cada uma identificada por um nome único. Podemos colocar diferentes valores nessas caixas ao longo do tempo conforme necessário durante a execução do programa. No entanto, é importante notar que uma variável só pode conter um valor por vez. Assim, mesmo que possa mudar ao longo do tempo, em qualquer instante durante a execução do programa, uma variável só guarda um único valor.

Constante

É um valor fixo, que não se altera ao longo do programa. Por exemplo, você pode definir uma constante chamada "PI" com o valor 3.14159. Mesmo que o programa faça cálculos com esse valor, ele permanecerá constante e não será alterado.

Numéricos

  • Inteiro

    • Variáveis do tipo inteiro são utilizadas para armazenar números inteiros, ou seja, números que não possuem parte fracionária. Em Pascal, essas variáveis são chamadas de 'Integer'. Elas podem representar números positivos, negativos ou zero. Exemplos comuns de números inteiros incluem -5, 0, 10 e 1000.

  • Real

    • Os números reais, também chamados de números de ponto flutuante ou números decimais, são um tipo de variável usado em programação para representar valores com parte fracionária. Em Pascal, esse tipo de variável é denominado 'Real'.

    Os números reais podem ser positivos, negativos ou zero, e podem incluir frações decimais. Por exemplo, 3.14, -0.5 e 1000.99 são todos exemplos de números reais.

    No sistema de escrita comum, o ponto "." é utilizado para separar a parte inteira da parte decimal de um número real.

Tipos de Variáveis - Char e String

Caracteres

Imagine que você queira guardar uma letra ou um símbolo, como uma letra do alfabeto, um número de telefone, ou um caractere especial, como '@' ou '!'. Em Pascal, usamos o tipo de variável 'char' para isso. Por exemplo, podemos criar uma variável chamada 'letra' e guardar nela a letra 'A'.

AlfaNuméricos

Já os alfanuméricos são uma mistura de letras e números. Por exemplo, um nome ou um endereço de e-mail. Em Pascal, podemos usar uma sequência de caracteres para guardar essas informações, geralmente utilizando o tipo de variável 'string'.

Tipos de Variáveis - Lógicas

Lógicas

Em Pascal, temos um tipo especial chamado 'booleano' que guarda só duas coisas: verdadeiro ou falso. Por exemplo, podemos criar uma variável chamada 'chovendo' e colocar nela 'verdadeiro' se estiver chovendo ou 'falso' se não estiver.

Isso ajuda o programa a entender o que está acontecendo no mundo real. Isso torna os programas capazes de lidar com muitos tipos diferentes de informações, o que é bem útil!

Como Iniciar um Pseudocódigo

Comece com Algoritmo Nome_do_Algoritmo;

Não se esqueça que ao todo final de linha de comando tem que usar o ";" . Ele serve pra dizer que aquele comando já foi finalizado

Explique para que serve o algoritmo dentro de chaves, como {Explique para que serve o algoritmo}.

O "algoritmo dentro de chaves" é uma forma de comentário no pseudocódigo, utilizado para explicar o propósito ou a função de determinada parte do código. Esses comentários são ignorados pelo interpretador do pseudocódigo e servem apenas para facilitar a compreensão do algoritmo pelos programadores.

Lembre-se que a parte da explicação NÃO é linha de código, então não leva ";" no final.
