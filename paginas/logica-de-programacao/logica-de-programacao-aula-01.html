Lógica de Programação - 01

ALGORITMOS

LÓGICA DE PROGRAMAÇÃO:

É como pensamos para resolver um problema. Imagine isso como seguir uma receita de bolo. Você precisa entender cada passo e organizá-los de forma lógica para que o bolo saia certo no final. Isso envolve saber usar os ingredientes (dados) da maneira certa e seguir a ordem correta dos passos.

ALGORITMO:

Um algoritmo é uma sequência finita de instruções bem definidas e não ambíguas (não têm duplo sentido) que levam à solução de um problema específico. É uma descrição precisa e ordenada de como realizar ou resolver uma tarefa. Um algoritmo pode ser implementado em qualquer linguagem de programação e é independente de uma linguagem específica.

Os algoritmos são extremamente necessários para a programação, pois são a base para a implementação de soluções em formato de código.

PSEUDOCÓDIGO :

É como dar instruções para resolver um problema usando uma linguagem simples que todos entendem, sem se preocupar com as regras de programação. É como fazer um esboço de um plano antes de começar a fazer algo. Você descreve passo a passo o que precisa ser feito de forma clara e direta, ajudando a entender como resolver o problema antes de escrever o código real em uma linguagem de programação específica. Os algoritmos que vamos trabalhar vão ser feitos em pseudocódigos.

Regras para Criar um Algoritmo

  • Usar somente um verbo por frase(Um comando)
  • O objetivo precisa ser claro
  • Usar frases curtas e simples
  • Não utilizar palavras com duplo sentido

Componentes Básicos de um algoritmo

  • Dados de entrada: São os dados que o usuário vai digitar. São os dados necessários para que o programa seja executado. São variáveis.
  • Dados de processamento: São aqueles que o algoritmo "manipula" ou "trabalha" durante sua execução para produzir o resultado desejado.
  • Dados de saída: são os resultados ou informações que o algoritmo produz após processar os dados de entrada. Eles representam a resposta ou o resultado final do algoritmo com base nas operações realizadas.

    • Exemplo-de-entrada/processamento/saida - MEDIA

  • Entrada: Escolha três números.
  • Processamento: Some os três números.
    Divida a soma pelo número total de números (que é 3 neste caso).
  • Saída: O resultado do cálculo é a média dos três números.
